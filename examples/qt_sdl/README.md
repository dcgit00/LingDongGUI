# Qt SDL Demo的使用

#### 安装qt

1、选择mingw编译器。

2、双击Virtual_TFT.pro，启动Qt。

3、项目打开后，会自动复制SDL2的dll文件，到对于的文件内，无需手动操作。

4、直接点击左下角三角形，进行编译、运行。

#### 注意

arm-2d的branch是developing，请按实际情况切换。
