## 扩展性

灵动GUI是基于ARM-2D进行开发的GUI

* 极大降低ARM-2D的使用难度

* 使用ARM-2D API自由组合控件

## 通用性

可以通过模拟器在PC上运行，解除了只能在Cortex-M系列单片机上运行的限制

* 支持ARM单片机、RISC-V单片机

* 需要支持GNU 99以上编译器

## 易用性

* 使用类似Qt的信号槽模式

* 界面代码和逻辑代码分离
